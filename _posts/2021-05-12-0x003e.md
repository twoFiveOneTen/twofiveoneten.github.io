彩色的终端能帮助我更快地看清楚目录结构，怎么设置彩色的终端？<!--more-->
# bash
1.先切换到bash
```shell
chsh -s /bin/bash
```

2.更改.bash_profile配置文件
```shell
vim ~/.bash_profile
```
添加以下内容
```shell
export CLICOLOR=1
export LSCOLORS=ExGxBxDxCxEgEdxbxgxcxd
```
`LSCOLORS`颜色配置方法请参考[Geoff.Greer.fm](https://geoff.greer.fm/lscolors/ "Geoff.Greer.fm")

更改完成后使其生效
```shell
source ~/.bash_profile
```
